优酷视频地址解析

算法参考于 优酷真实视频地址解析——2014年10月7日

代码部分来源于 Video-Downloader

代码于2016年3月14日运行可行

1.获取视频vid

eg：http://v.youku.com/v_show/id_**XMTQ5MTk2MDI1Mg==**.html

从视频网址可以得出视频的vid：XMTQ5MTk2MDI1Mg==

利用正则表达式获取

`def __getVideoID(self, link):
    result =re.findall(r"(?<=id_).*(?=.html)",link)
    if len(result) > 0 :
        videoID = result[0]
    else :
        videoID = False

    return videoID `

2.获取视频信息

可根据抓包软件得到视频信息的入口：

http://play.youku.com/play/get.json?vid={vid}&ct=12

vid部分换成要解析视频网址的vid

得到一个json格式的文件

其中stream_type显示该视频含有的格式
该文件中有用的数据有ip，encrypt_string

3.计算ep，sid，token

def __getVideoFileUrl (self, videoInfo) :
    videoInfo = json.JSONDecoder().decode(videoInfo)
    if 'security' in videoInfo['data']:
        ep = videoInfo['data']['security']['encrypt_string']
        print("ep:"+ep)
    else :
        ep = False

    if ep :
        oip   = videoInfo['data']['security']['ip']
        vid   = videoInfo['data']['video']['encodeid']
        temp  = self.__yk_e('becaf9be', base64.decodestring(ep))
        sid   = temp.split('_')[0]
        token = temp.split('_')[1]
        ep    = urllib.quote(base64.encodestring(self.__yk_e('bf7e5f01', str(sid) + '_' + str(vid) + '_' + str(token))), '')[0:-3]
        fileUrl = self.fileUrlPrefix + '&ep=' + str(ep) + '&oip=' + str(oip) + '&sid=' + str(sid) + '&token=' + str(token) + '&vid=' + str(vid) + '&type=' + self.videoTypeList[self.videoType]+'&special=true'
    else :
        fileUrl = False

    print(fileUrl)

    return fileUrl

def __yk_e (self, a, c) :
    f = 0
    i = 0
    h = 0
    b = {}
    e = ''
    for h in xrange(0, 256) :
        b[h] = h;

    for h in xrange(0, 256) :
        f = ((f + b[h]) + self.__charCodeAt(a, h % len(a))) % 256;
        i = b[h]
        b[h] = b[f]
        b[f] = i

    f = h = 0
    for q in xrange(0, len(c)) :
        h = (h + 1) % 256
        f = (f + b[h]) % 256
        i = b[h]
        b[h] = b[f]
        b[f] = i
        e = e + self.__fromCharCode(self.__charCodeAt(c, q) ^ b[(b[h] + b[f]) % 256]);

    return e

4.拼接m3u8地址

http://pl.youku.com/playlist/m3u8?ctype=12&ep={0}&ev=1&keyframe=1&oip={1}&sid={2}&token={3}&type={4}&vid={5}

ep,sid,token由计算得出；type视频格式，可选hd2，mp4，flv等，oip，vid就是前面得到的ip和id
得出的m3u8截图如下：

其中每条url只包含6s左右视频，但是可将url中参数部分去掉即可得到实际的长度。但是每条去掉后需合并一下相同的url，如上述列表可得到url片段

http://103.41.140.136/67716E666BA497D6C738B2874/0300080F0056E0FDC17B5E132D24F5A868531E-E590-D25C-3CB0-1DA01801B465.mp4

将m3u8中的所有url片段全部下载即可大功告成


获取视频的时候可能会出现一个问题，m3u8文件显示的视频地址不全，一般只有前50分钟，后面的视频地址获取不到；

但是作者发现，如果优酷登录了会员账号，再利用浏览器的cookie访问同样的m3u8网址时就能获取视频的全部地址

cookie里面唯一需要的字段：ykss
